Mon projet étant de réussir le cours "Gérez votre code avec Git et Github" sur Openclassrooms, voici un petit aperçu de quoi est composé l'activité que l'on nous demandait de réussir pour obtenir le certificat de réussite :
Il nous fallait créer un répertoire contenant un fichier README.txt qui est donc celui que vous lisez en ce moment même, puis créer au moins deux autres fichiers et au moins 4 commits distincts.
Voilà vous savez tout, à vous de juger si je remplis les conditions pour la réussite de ce cours ou non.
 A bonne entendeur, salut.
